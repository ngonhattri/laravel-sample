## Note

1. サーバーの起動

```
php artisan serve
```

2. モデルとコントローラ一緒に作成

```
php artisan make:model モデル名 -r
```

3. Laravel のコンソールの使用

```
php artisan tinker
```

-   PHP Error: Class "..." not found の場合　->　 namespace 変更

```
>>> namespace App\Models

```
4. マイグレーションの生成->新規テーブルを作成

```
php artisan make:migration テーブル名
```

5. Migration

```
php artisan migration
```

6. Seeder

```
php artisan db:seed
```
